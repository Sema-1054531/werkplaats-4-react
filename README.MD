# werkplaats-4-react-one-piece_v2
Benodigde software: Node.js

Om React server te starten: 
``` 
cd one_piece
npm start
```

Om de database te laden: 
``` 
Open een tweede terminal.
cd one_piece/server
node server.js
```

Om de server af te sluiten: 
``` 
Druk op Ctrl-C.
Druk op Y bij de vraag en druk op enter.
```
